## Overview

The Assessment Results data product is the canonical scored output of the ases.id measurement engine. It represents the transformation of raw assessment responses into structured Core Behaviour scores and fit metrics.

Consumers (recru.id, peningkatan.id) gate access via the bakat.id consent model — a person must have active `ConsentGranted` before their assessment result can be read cross-domain.

## Access model

| Consumer         | Access condition                                        |
|------------------|---------------------------------------------------------|
| recru.id         | Same-tenant OR person has active ConsentGranted         |
| peningkatan.id   | Same-tenant OR person has active ConsentGranted         |
| bakat.id         | Always — it owns the person's canonical profile         |

## Freshness

Updated within seconds of `AssessmentCompleted` via the ScoringService event handler.
