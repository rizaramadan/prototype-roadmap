## Overview

The Core Behaviour Profile is the central data product of the Talentlytica ecosystem. Every scored assessment contributes to this profile. Unlike an ATS resume, it belongs to the **person**, not the employer — and it travels with them across organizations.

**Portability:** When a person joins a new employer, they authorize that employer (via `GrantConsent`) to read their profile. The profile is never copied — cross-tenant reads go through bakat.id's consent-gated API.

**Longitudinal:** Each assessment appends a new data point. The profile shows growth, regression, and stability across time and contexts (different roles, different employers).

## Consent gates

| Operation      | Condition                                     |
|----------------|-----------------------------------------------|
| Read (same tenant) | Always allowed                            |
| Read (cross-tenant) | Active `ConsentGranted` required          |
| Write          | Only ScoringService via internal event        |

## Freshness

Updated within seconds of every `AssessmentCompleted` event via `PersonCoreBehaviourProfileUpdated`.
