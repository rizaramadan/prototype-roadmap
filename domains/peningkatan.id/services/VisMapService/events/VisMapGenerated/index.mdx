Emitted when a vis map computation completes. Contains the overall gap score and per-Core-Behaviour deltas. IDPService consumes this event to recommend targeted interventions.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field           | Type     | Notes                                                      |
|-----------------|----------|------------------------------------------------------------|
| visMapId        | uuid     | Unique identifier for this map snapshot                    |
| personId        | uuid     | bakat.id person                                            |
| positionId      | uuid     | Target position                                            |
| tenantId        | string   | Requesting organization                                    |
| overallGapScore | number   | 0.0 (perfect fit) – 1.0 (maximum gap)                     |
| gaps            | object[] | Per-Core-Behaviour gap: `{ coreBehaviourId, required, actual, delta }` |
| generatedAt     | datetime | ISO 8601                                                   |
