---
id: CompetencyUpdated
name: CompetencyUpdated
version: 1.0.0
summary: A competency's name, description, or Core Behaviour membership has changed
owners:
  - platform
---
Emitted after a successful competency update. Downstream systems that cache competency definitions must invalidate on this event.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field                   | Type     | Notes                                              |
|-------------------------|----------|----------------------------------------------------|
| competencyId            | uuid     | Updated competency                                 |
| name                    | string?  | New name if changed                                |
| addedCoreBehaviourIds   | uuid[]?  | Core Behaviours added in this update               |
| removedCoreBehaviourIds | uuid[]?  | Core Behaviours removed in this update             |
| updatedAt               | datetime | ISO 8601                                           |
