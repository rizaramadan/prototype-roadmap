Emitted when an award is successfully recorded. The award is visible on the person's portable bakat.id profile and may trigger notifications to the recipient.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field              | Type     | Notes                                             |
|--------------------|----------|---------------------------------------------------|
| awardId            | uuid     | Unique identifier for this award record           |
| personId           | uuid     | bakat.id identifier of the recipient              |
| awardType          | enum     | top_performer \| learning_agility \| consistency  |
| sourceAssessmentId | uuid?    | Assessment that triggered this award              |
| tenantId           | string   | Issuing organization                              |
| issuedAt           | datetime | ISO 8601                                          |
