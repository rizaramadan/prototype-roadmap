Emitted when a returning person opts into the "use previous data" path during intake. The new assessment inherits intake answers from the referenced prior assessment, skipping the intake form entirely.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field                | Type     | Notes                                          |
|----------------------|----------|------------------------------------------------|
| assessmentId         | uuid     | New assessment being started                   |
| previousAssessmentId | uuid     | Source assessment whose intake data is reused  |
| personId             | uuid     | bakat.id person identifier                     |
| initiatedAt          | datetime | ISO 8601                                       |
