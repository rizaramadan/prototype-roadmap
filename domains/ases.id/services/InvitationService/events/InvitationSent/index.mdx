Assessment invitation email has been sent. Downstream services use this event to track invitation delivery state and enforce re-invite cooldown windows.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field        | Type     | Notes                               |
|--------------|----------|-------------------------------------|
| invitationId | uuid     | Unique identifier for this invite   |
| assessmentId | uuid     | Assessment the invite links to      |
| email        | string   | Recipient address                   |
| sentAt       | datetime | ISO 8601                            |
| locale       | string?  | Language the email was sent in      |
