Emitted after ATSBridgeService successfully creates or updates a candidate in CandidateService. Provides the cross-reference between the external ATS ID and the internal candidateId for downstream traceability.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Schema

| Field       | Type     | Notes                                              |
|-------------|----------|----------------------------------------------------|
| candidateId | uuid     | Internal recru.id candidate identifier             |
| atsSource   | enum     | workday \| greenhouse \| lever \| successfactors   |
| atsId       | string   | Candidate ID in the external system                |
| tenantId    | string   | Hiring organization                                |
| syncedAt    | datetime | ISO 8601                                           |
