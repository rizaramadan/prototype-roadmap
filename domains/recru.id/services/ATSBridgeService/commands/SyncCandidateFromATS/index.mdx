Issued by: **ATSBridgeService (internal webhook handler)**

Triggered when an external ATS webhook pushes a new applicant event. ATSBridgeService translates the external payload into the recru.id canonical model and emits a `SyncCandidateFromATS` command to itself, which then creates or updates the candidate via CandidateService.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Payload

| Field         | Type   | Notes                                              |
|---------------|--------|----------------------------------------------------|
| atsSource     | enum   | workday \| greenhouse \| lever \| successfactors   |
| atsId         | string | Candidate ID in the external system                |
| email         | string | Candidate email (used for deduplication)           |
| jobRef        | string | External job reference                             |
| tenantId      | string | Hiring organization                                |
| rawPayload    | object | Original ATS payload stored verbatim for auditing  |
