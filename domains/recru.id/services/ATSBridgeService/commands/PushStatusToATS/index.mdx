  path: /ats/push/status

Issued by: **ATSBridgeService (internal, triggered by CandidateHired / CandidateRejected / AssessmentCompleted events)**

Translates recru.id domain events into the external ATS's API format and pushes the update. Retries on 503. Failures are dead-lettered for manual review.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Payload

| Field     | Type   | Notes                                              |
|-----------|--------|----------------------------------------------------|
| atsSource | enum   | workday \| greenhouse \| lever \| successfactors   |
| atsId     | string | Candidate ID in the external system                |
| status    | enum   | hired \| rejected \| assessment_completed          |
| reason    | string? | Rejection reason if status = rejected             |
| tenantId  | string | Hiring organization                                |
