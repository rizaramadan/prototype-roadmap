Emitted once AssessmentDelegationService has received confirmation from IngestionService that the assessment was created (`AssessmentAssigned`). ApplicationService consumes this to track that the candidate is awaiting assessment.

| Field         | Type   | Notes                                                    |

| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |
