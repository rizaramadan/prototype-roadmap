Issued by: **recru.id recruiter (via API Gateway)**

Instructs AssessmentDelegationService to coordinate with IngestionService (via `AssignAssessment`) to create a candidate-specific assessment instance. The delegation is idempotent — re-submitting with the same `candidateId` + `jobId` returns the existing delegation.

## Envelope

| Field         | Type   | Notes                                                    |
|---------------|--------|----------------------------------------------------------|
| traceId       | uuid   | Propagated from originating request                      |
| correlationId | uuid   | Groups related events across services and workflows      |
| causationId   | uuid?  | ID of the event or command that directly caused this one |

## Payload

| Field              | Type   | Notes                                          |
|--------------------|--------|------------------------------------------------|
| candidateId        | uuid   | Candidate in the recruitment funnel            |
| jobId              | uuid   | Job the candidate applied for                  |
| assessmentConfigId | uuid   | Which assessment battery to use                |
| tenantId           | string | Hiring organization                            |
