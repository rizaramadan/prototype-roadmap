Emitted when a student enrolment is successfully recorded. The beasiswa.bakat.id domain consumes this to evaluate scholarship eligibility.

| causationId   | uuid?  | ID of the event or command that directly caused this one |

| Field         | Type     | Notes                                        |
|---------------|----------|----------------------------------------------|
| studentId     | string   | Institution-assigned student ID              |
| personId      | uuid     | bakat.id person identifier                   |
| institutionId | uuid     | Enrolling institution                        |
| programId     | uuid     | Degree or certificate program                |
| enrolledAt    | datetime | ISO 8601                                     |
